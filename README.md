# aramb Homebrew Tap

Official Homebrew tap for [aramb-cli](https://github.com/aramb-ai/aramb-cli).

## Install

```sh
brew tap aramb-ai/tap
brew install aramb
```

## Upgrade

```sh
brew upgrade aramb
```

## Uninstall

```sh
brew uninstall aramb
brew untap aramb-ai/tap
```

## Supported Platforms

| OS    | Architecture |
|-------|-------------|
| macOS | arm64 (Apple Silicon) |
| macOS | amd64 (Intel) |
| Linux | arm64 |
| Linux | amd64 |
